Clear
=====
The Clear function is used to remove all events currently contained in the 
event storage.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrhpsb/MRH_EventStorage.h>


Syntax
------
.. code-block:: c

    void Clear() noexcept;


Parameters
----------
None.

Return Value
------------
None.

Remarks
-------
* This function is thread safe.

Code Examples
-------------
None.
