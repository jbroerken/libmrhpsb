MRH_GetLibraryVersionMinor
==========================
The MRH_GetLibraryVersionMinor function is used to obtain the minor library 
version.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrhpsb/MRH_PSBRevision.h>


Syntax
------
.. code-block:: c

    int MRH_GetLibraryVersionMinor() noexcept;


Parameters
----------
None.

Return Value
------------
The minor library version defined by 
:doc:`../../../Macros/libmrhpsb_Directory/MRH_PSBRevision/MRH_PLATFORM_SERVICE_BASE_LIB_VERSION_MINOR`.

-------
None.
