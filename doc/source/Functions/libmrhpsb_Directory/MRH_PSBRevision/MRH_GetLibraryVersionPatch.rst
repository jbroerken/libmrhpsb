MRH_GetLibraryVersionPatch
==========================
The MRH_GetLibraryVersionPatch function is used to obtain the library patch 
version.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrhpsb/MRH_PSBRevision.h>


Syntax
------
.. code-block:: c

    int MRH_GetLibraryVersionPatch() noexcept;


Parameters
----------
None.

Return Value
------------
The library patch version defined by 
:doc:`../../../Macros/libmrhpsb_Directory/MRH_PSBRevision/MRH_PLATFORM_SERVICE_BASE_LIB_VERSION_PATCH`.
